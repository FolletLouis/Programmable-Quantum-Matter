# Static assets for paper explorer

Add logo files here to display in the app header:

- **mit_logo.png** — MIT logo (recommended size: ~200×80 px or similar aspect)
- **eth_logo.png** — ETH Zurich logo (recommended size: ~200×80 px or similar aspect)

Download from official sources:
- MIT: https://brand.mit.edu/logos-marks/mit-logo
- ETH Zurich: https://ethz.ch/staffnet/en/service/communication/corporate-design/logo.html

If these files are missing, the app shows text placeholders (MIT / ETH Zürich).
